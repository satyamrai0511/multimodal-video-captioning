# 🎥 Multimodal Video Captioning using Diffusion and Autoregressive Models

This is a portfolio project that combines **diffusion models** and **multimodal autoregressive architectures** to generate context-aware and temporally coherent captions from video and audio streams.

## 📌 Project Goals

- Generate descriptive captions from video + audio input.
- Fuse visual and auditory features for better context understanding.
- Improve temporal coherence and linguistic fluency in generated text.

## 📁 Folder Structure
```
├── data/           # Placeholder or instructions for datasets
├── models/         # Pretrained or saved models
├── notebooks/      # Jupyter notebooks for development and demo
├── scripts/        # Python scripts for training, inference
├── utils/          # Helper functions for preprocessing, loading data
├── requirements.txt
├── README.md
├── .gitignore
```

## ⚙️ Tech Stack

- Python, PyTorch, OpenCV, Librosa
- Transformers (HuggingFace), DDPM (Diffusion)
- Jupyter, Streamlit (optional for demo)

## 🧠 Core Concepts

- **Diffusion Models** for denoising + feature enhancement
- **Autoregressive Language Models** for caption generation
- **Multimodal Fusion**: combining video frames and audio features
